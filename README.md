# SetUp the script

 - ```  $ git clone https://github.com/cosimochellini/automazioneRimborsoSpese.git```
 - ```  $ npm install``` 
 - ```  $ cd automazioneRimborsoSpese```
  - Rename the settings.example.json in settings.json
  - Edit the settings.json 
  - ```  $ node run.js```
